# Docs

Разработано: **Цветков Артём Денисович**, **СГТУ им. Гагарина Ю.А.**

Папка содержит документацию, которую можно использовать при описании тестирования и архитектуры проекта в магистерской работе.

## Файлы

- `project_structure.md` — актуальная структура репозитория.
- `test_scenarios.md` — end-to-end сценарии проверки backend/frontend и бизнес-формул.
- `paper_ru.md` — русская статья-описание проекта для магистерской работы.
- `paper_ru.pdf` — PDF-версия русской статьи.

Для текущей магистерской работы рекомендуется использовать русскую версию `paper_ru.md` или `paper_ru.pdf`.

Документы в этой папке не участвуют в запуске приложения и не изменяют модель.
