# Helper Files

Разработано: **Цветков Артём Денисович**, **СГТУ им. Гагарина Ю.А.**

Архивная папка со вспомогательными скриптами визуализации и анализа данных.

Содержит ранние проверки:

- распределение бюджета и кассы;
- гистограммы;
- проверка пропусков;
- диаграммы;
- анализ значимости признаков.

Для актуального исследования рекомендуется использовать ноутбуки из `notebooks/`.
