# Old Datasets

Разработано: **Цветков Артём Денисович**, **СГТУ им. Гагарина Ю.А.**

Папка содержит архивные, промежуточные и исторические датасеты, которые использовались на ранних этапах исследования.

Эта папка не используется для финального production-обучения. Основной датасет проекта находится здесь:

```text
data/processed/output.csv
```

## Назначение

Материалы в `old datasets/` можно использовать для описания эволюции исследования:

- первичные объединения разных источников;
- промежуточные CSV;
- Kaggle-выгрузки;
- ранние попытки сборки единого набора данных.

## Важно

Не рекомендуется смешивать эти файлы с production pipeline без отдельной проверки схемы, пропусков, дублей и лицензий источников.
