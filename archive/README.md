# Archive

Разработано: **Цветков Артём Денисович**, **СГТУ им. Гагарина Ю.А.**

Папка содержит материалы ранних экспериментов, которые не входят в основной production-контур веб-сервиса.

## Состав

- `legacy_apps/` — старые CLI/Streamlit-прототипы.
- `legacy_models/` — ранние исследовательские скрипты моделей.
- `legacy_helpers/` — вспомогательные скрипты анализа и визуализации.
- `legacy_datasets/` — архивные и промежуточные датасеты.
- `legacy_misc/` — черновые материалы.

Актуальный рабочий контур проекта находится в `artifacts/`, `backend/`, `frontend/`, `ml_pipeline/`, `scripts/`, `notebooks/`, `data/` и `docs/`.
